# mst team list Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Implement `mst team list` — navigates to teams.microsoft.com, intercepts the internal API response, and outputs all joined teams as a JSON envelope to stdout.

**Architecture:** A new `src/scrapers/teams.ts` module exports `listTeams(session)`, which uses the `withBrowser` HOF to navigate to `teams.microsoft.com` and intercept the `/api/csa/{region}/api/v3/teams/users/me` response. The CLI loads the session via `ensureValidSession()` and passes it to `listTeams()`, then formats and prints the result.

**Tech Stack:** TypeScript, Playwright, Vitest, commander

## Global Constraints

- Node 24.18.0 (pinned via `.nvmrc`)
- pnpm 11.9.0 — use `pnpm` not `npm`
- All dependencies pinned exactly (no `^` ranges)
- Tests use Vitest (`pnpm test`)
- Lint: `pnpm lint` (oxlint), Format: `pnpm format` (oxfmt)
- No Graph API usage
- All raw commands output `{ success, data, metadata }` JSON to stdout

```ts
import type { BrowserContext } from "playwright";
import { withBrowser } from "../browser.js";

type StorageState = Awaited<ReturnType<BrowserContext["storageState"]>>;

export type Team = {
  id: string;
  displayName: string;
  description: string | null;
};

type RawTeam = {
  id: string;
  displayName: string;
  description?: string | null;
  isDeleted?: boolean;
};

type TeamsResponse = {
  teams: RawTeam[];
};

const TEAMS_URL_PATTERN = /\/api\/csa\/.+\/api\/v3\/teams\/users\/me/;

export async function listTeams(session: StorageState): Promise<Team[]> {
  return withBrowser(session, async ({ page }) => {
    const responsePromise = page.waitForResponse(TEAMS_URL_PATTERN, { timeout: 60_000 });
    await page.goto("https://teams.microsoft.com");
    const response = await responsePromise;
    const body = (await response.json()) as TeamsResponse;
    return body.teams
      .filter((t) => !t.isDeleted)
      .map((t) => ({
        id: t.id,
        displayName: t.displayName,
        description: t.description ?? null,
      }));
  });
}
```

const team = program.command("team");

team
  .command("list")
  .description("List all joined teams")
  .action(async () => {
    const start = Date.now();
    const session = await ensureValidSession();
    const teams = await listTeams(session);
    const durationMs = Date.now() - start;
    console.log(JSON.stringify(successEnvelope({ teams }, durationMs)));
    if (process.stdout.isTTY) {
      for (const t of teams) {
        process.stderr.write(`${t.displayName}\t${t.id}\n`);
      }
    }
  });
